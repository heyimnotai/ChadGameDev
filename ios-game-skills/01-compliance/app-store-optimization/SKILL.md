# App Store Optimization

## Purpose

Maximize app visibility and conversion through strategic metadata, keywords, and visual assets that drive downloads.

## When to Use

- Writing app title, subtitle, description
- Researching keywords for your game genre
- Planning screenshot sequence
- Preparing for launch or major update
- Conversion rate is below 30%

## Core Process

1. **Keyword Research**: Identify 10-15 relevant keywords with search volume
2. **Title Optimization**: Game name + primary keyword (30 chars max)
3. **Subtitle**: Secondary value prop + keyword (30 chars max)
4. **Keyword Field**: 100 chars, comma-separated, no spaces after commas
5. **Description**: Hook in first 167 chars (visible without tap)
6. **Screenshots**: First 3 are critical - show core gameplay + value
7. **A/B Test**: Use App Store Connect product page optimization

## Key Rules

**Title/Subtitle**:
- No keyword stuffing
- Must reflect actual game
- Avoid trademarked terms

**Keywords**:
- No duplicates from title/subtitle (waste)
- Singular, not plural
- No category names (Apple ignores)
- Competitor names = rejection risk

**Screenshots**:
- First 3 visible without scroll - make them count
- Show actual gameplay, not just splash screens
- Text overlays: benefits, not features
- Device frames optional but consistent

## Quick Reference

| Element | Limit | Priority |
