# Core Expander - API Reference

## Game Analysis Framework

| Dimension | Questions |
|-----------|-----------|
| Genre | Action, puzzle, idle, strategy, RPG, casual? |
| Session Length | 30 seconds, 2 minutes, 10+ minutes? |
| Progression | Level-based, endless, campaign? |
| Player Agency | High (strategy), medium (action), low (idle)? |
| Social | Solo, competitive, cooperative? |
| Monetization | Free, premium, freemium? |
| Current Depth | What mechanics exist? |

## Mechanic Categories

| Category | Best For | Examples |
|----------|----------|----------|
| Progression & Meta | Long-term retention | Prestige, Battle Pass, Mastery |
| Loot & Rewards | Engagement spikes | Chests, Gacha, Daily Rewards |
| Economy & Trading | Depth + Monetization | Shops, Currencies, Crafting |
| Character Systems | Identity + Collection | Traits, Classes, Skill Trees |
| Combat & Enemies | Challenge variety | Enemy Types, Bosses, AI |
| Collection & Items | Completionism | Equipment, Collectibles |
| Social & Multiplayer | Virality | Leaderboards, Guilds, PvP |
| Engagement Hooks | Daily retention | Achievements, Quests, Streaks |
| Strategic Depth | Skill expression | Deck Building, Resource Mgmt |
| Unique & Emergent | Differentiation | Roguelike, Procedural |

## Priority Matrix

| Factor | Weight | Evaluation |
|--------|--------|------------|
| Genre Fit | 25% | Matches player expectations? |
| Effort Required | 25% | Days/weeks/months to build |
| Retention Impact | 20% | D1/D7/D30 lift expected |
| Synergy | 15% | Multiplies existing mechanics? |
| Uniqueness | 15% | Differentiates from competitors? |

## Implementation Phases

| Phase | Focus | Max Mechanics |
|-------|-------|---------------|
| 1: Foundation | Core expansion | 2-3 |
| 2: Depth | Long-term hooks | 2-3 |
| 3: Polish | Social + Meta | 2-3 |
| 4: Differentiation | Unique features | 1-2 |

## Genre-Specific Must-Haves

### Idle Games
- Prestige System (+10-100x playtime)
- Offline Progress (reduces churn)
- Upgrade Trees (meaningful choices)

### Puzzle Games
- Star Rating (replayability)
- Power-ups (strategic depth)
- Episode Gates (pacing)

### Action/Arcade
- Leaderboards (competition)
- Combo Systems (skill expression)
- Unlockable Characters (variety)

### Strategy/Tower Defense
- Unit Upgrades (progression)
- Star Rating (mastery)
- Multiple Currencies (depth)

### Roguelike
- Meta Progression ("one more run")
- Random Item Pool (discovery)
- Risk/Reward Choices (tension)

## Weakness-Based Recommendations

| Weakness | Add |
|----------|-----|
| Session too short | Combo, Unlockables, Challenges |
| Players leave D3-7 | Daily Rewards, Streaks, Quests |
| Players finish content | Prestige, Endless Mode, Procedural |
| Low virality | Leaderboards, Sharing, Guilds |
| Low monetization | Battle Pass, Shop, Premium Currency |
