## Pity System Types

| Type | Mechanic | Best For |
|------|----------|----------|
| Hard Pity | Guaranteed at N pulls | Frustration prevention |
| Soft Pity | Increasing rate after N | Natural anticipation |
| Spark/Select | Choose after N pulls | Player agency |
