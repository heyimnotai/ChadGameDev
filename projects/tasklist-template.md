# Game TaskList

> Fill this file with bugs to fix and features to add. Chad will work through each item until all are complete.

## Format

Use checkboxes for each task. Chad will mark them `[x]` when complete.

```markdown
- [ ] Bug: Description of the bug
- [ ] Feature: Description of the feature to add
- [ ] Polish: Description of the polish/juice to add
- [ ] Refactor: Description of code to clean up
```

## Priority Tags (Optional)

Add priority tags to control order:

- `[P0]` - Critical (blocking, crashes)
- `[P1]` - High (major bugs)
- `[P2]` - Medium (minor bugs)
- `[P3]` - Low (polish, nice-to-have)

Example: `- [ ] [P0] Bug: Game crashes when score reaches 1000`
